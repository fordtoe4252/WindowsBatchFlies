# Alexander's Windows Toolbox

A collection of safe, admin-only Windows BAT utilities created by Alexander.

## Features
- Device info
- Network tools
- Internet ping test
- User account info
- Disk info
- System uptime
- And more...
Note:These Bat Flies Are Safe And Do Not Change Your Settings.
Another Note:Even Though I Said Safe, The Bat Flies Are Still In Beta, So, Be Careful. Also, Im Not Responsible For Any Damages Made To Your Device.
